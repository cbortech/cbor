Convert between CBOR, CDN (CBOR-EDN), and JavaScript values

@cbortech/cbor is a zero-dependency TypeScript library implementing RFC 8949 CBOR and CDN (CBOR-EDN) draft-25 / draft-26 — encode, decode, parse, pretty-print, and round-trip with full fidelity. Everything on this page runs in your browser using the library itself.

npm install @cbortech/cbor
  • Lossless round-trips — encoding widths and comments survive CDN (CBOR-EDN) ⇄ CBOR conversion.
  • JSON-like API — CBOR.parse / CBOR.stringify with reviver and replacer support.
  • AST access — inspect every data item with byte ranges, warnings, and annotated hex dumps.
  • Extensible — application-oriented extensions for dates, IP addresses, string concatenation, and more, plus optional hash support. Toggle any extension from the Extensions button in the playground control bar below.
